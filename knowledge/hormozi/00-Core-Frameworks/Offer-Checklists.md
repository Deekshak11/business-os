# Offer Checklists (Companion)

Use as **audit prompts** while building offers. Pair with full frameworks.

## Niche / market checklist

- [ ] Clear avatar with money + pain  
- [ ] Problem is urgent / costly if unsolved  
- [ ] Can reach them via Core Four  
- [ ] Message is specific (person + problem + context)  

## Offer creation checklist

- [ ] Dream outcome written in customer language  
- [ ] Problems list between them and dream  
- [ ] Solutions/bonuses for each major problem  
- [ ] Delivery vehicle chosen (DIY / DWY / DFY)  
- [ ] Trim low-value high-cost items  
- [ ] Value Equation bottoms reduced (speed, ease)  

## Pricing / value checklist

- [ ] Not competing as commodity on price alone  
- [ ] Premium justified by stack + proof + guarantee  
- [ ] Margin supports ads and fulfillment  
- [ ] Price tested against willingness + close rate  

## Bonus creation checklist

- [ ] Each bonus removes a real obstacle  
- [ ] Named and value-anchored  
- [ ] Delivery cost controlled  
- [ ] Order of presentation planned  

## Scarcity / urgency checklist

- [ ] Scarcity is **true** (capacity, cohort, bonus stock)  
- [ ] Urgency is **true** (deadline with reason)  
- [ ] Stated clearly without fake countdown abuse  

## Guarantee checklist

- [ ] Addresses main fear  
- [ ] Terms clear  
- [ ] Operationally deliverable  
- [ ] Increases perceived likelihood without bankrupting you  

## Product naming checklist

- [ ] Specific > generic  
- [ ] Outcome or mechanism visible  
- [ ] Hard to commodity-compare by name alone  
- [ ] Consistent with brand
