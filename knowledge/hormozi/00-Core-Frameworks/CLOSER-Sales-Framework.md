# CLOSER Sales Framework

## When to use

- Founder is “winging” sales calls  
- Stage 2 requirement: set + close script  
- Training first setters/closers  
- Need consistent discovery → offer → decision flow  

## Framework (CLOSER)

| Letter | Step | Purpose |
|--------|------|---------|
| **C** | **Clarify** why they’re here | Agenda, source, what they want help with |
| **L** | **Label** their problem | Reflect pain/problem clearly in their words |
| **O** | **Overview** past experiences | What they’ve tried; why it failed (builds gap) |
| **S** | **Sell** the vacation | Paint dream outcome / life after (not feature dump) |
| **E** | **Explain** away concerns | Objections, risk, fit; use proof + guarantee |
| **R** | **Reinforce** the decision | Next steps, onboarding, commitment, prevent buyer’s remorse |

> Exact phrasing varies by offer; structure stays stable. Deep scripts: `Playbook-Closing.md`.

## Operator rules

1. Diagnose before pitching.  
2. Sell the **outcome**, fulfill with the **vehicle**.  
3. Qualify (especially Stage 3+): wrong leads waste close skill.  
4. Record calls; improve one step per week.  
5. Align promises with CS (Stage 5 game tape).
