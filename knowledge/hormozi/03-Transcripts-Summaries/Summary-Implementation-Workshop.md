# Summary — Implementation Workshop

## When to use

- Strategy is clear; execution is not  
- Overwhelm from too many frameworks  
- Need a weekly implementation order  

## Implementation principle

**Do not run every playbook at once.** Sequence by **stage constraint**:

1. Diagnose stage (`01-Scaling-Stages/`)  
2. Fix the **primary constraint** only  
3. Use one framework + one playbook  
4. Measure one leading metric for 7–14 days  
5. Then unlock the next constraint  

## Default 30-day implementation tracks

### Track A — Stage 0–1 (no reliable revenue)
1. Narrow problem + free lead magnet  
2. Get 10–20 people to use free  
3. Convert feedback → V1 paid Grand Slam Offer  
4. First paid sales (simple ask)  
5. Metric: paid customers + cash  

### Track B — Stage 2 (inconsistent leads)
1. Pick one Core Four channel  
2. Rule of 100 daily  
3. Hook testing weekly  
4. CLOSER + closing drills  
5. Metric: engaged leads/day + close rate  

### Track C — Stage 3 (small team chaos)
1. Write top product complaint fix  
2. Install onboarding  
3. Basic lead qualification  
4. Email/content nurture cadence  
5. Metric: time-to-value for new customers; close rate of qualified leads  

### Track D — Stage 4–5 (team + economics)
1. Niche avatar + specialized offer  
2. KPIs + CRM notes  
3. Money model: upsell/continuity or second product  
4. If paid ads: ad assembly volume  
5. Metric: LTV:CAC, churn, contribution margin  

### Track E — Stage 6–7 (optimize + organize)
1. Separate product owners/pipelines for each product  
2. Ad assembly factory (hooks × meat × CTA)  
3. Sales training cadence + cohort retention  
4. Second acquisition channel funded by referrals + scored nurture  
5. Metric: CAC trend, close consistency, churn by cohort, lead waste rate  

### Track F — Stage 8–9 (specialize + capitalize)
1. Product prune survey + deletion plan  
2. Brand messaging standardization  
3. Lead routing + specialized sales/CS  
4. Rising-star bet (build vs buy) + capital readiness checklist  
5. Metric: premium close rate, brand/lead quality, contribution by division  

## Workshop-style working session agenda

1. Current numbers (headcount, leads, close, churn, cash)  
2. Stage declaration  
3. Constraint declaration  
4. 14-day action list (max 5 tasks)  
5. Owners + deadlines  
6. Review date
