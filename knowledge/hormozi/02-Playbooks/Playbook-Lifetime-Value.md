# Playbook — Lifetime Value

## When to use

- Ads only work at tiny spend  
- Single SKU caps customer spend  
- Churn destroys cohort value  
- Planning second product / continuity  

## LTV mental model

Lifetime value rises when you improve:

1. **Price** per transaction  
2. **Frequency** of purchase  
3. **Duration** / retention (lower churn)  
4. **Expansion** revenue (upsells, cross-sells, continuity)  

Rough: more cash over the customer’s life → higher allowable CAC → more growth.
