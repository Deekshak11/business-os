# Playbook — Marketing Machine

## When to use

- Marketing only happens when founder panics  
- No weekly creative/lead/sales loop  
- Want employees/agencies to run Core Four  
- Stage 4–5 process maturity  

## Machine definition

A marketing machine has:

1. **Inputs** — budget, time, hooks, offers, lists  
2. **Processes** — Core Four activities, ad assembly, nurture, sales handoff  
3. **Outputs** — engaged leads, appointments, customers, cash  
4. **Feedback** — KPIs that change next week’s inputs  

Without feedback, you have activity — not a machine.

## Build order (vault-aligned)

1. Grand Slam Offer + Lead Magnet  
2. Pick Core Four channels  
3. Rule of 100 volume (Stage 2)  
4. Hooks + (if paid) Goated Ads assembly  
5. Nurture + Closing  
6. Money model to fund the machine  
7. Stage-appropriate KPIs/CRM  

## Weekly operating rhythm (template)

| Day/Block | Focus |
|-----------|--------|
| Creative | Hooks / ads / content batch |
| Distribution | Outreach + posting + spend |
| Sales | Calls + follow-up |
| Review | KPIs: leads, shows, closes, CAC, cash |
| Fix | One constraint only |
