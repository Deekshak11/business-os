# Playbook — Price Raise

## When to use

- Costs up; price flat for years  
- Demand strong; capacity constrained  
- CAC requires higher LTV  
- After offer/value improvements  

## Core approach

1. **Earn the raise** — improve perceived value first (stack, results, speed, guarantee).  
2. **Segment** — new customers vs existing; often raise new first.  
3. **Grandfather policy** — decide who keeps old price and for how long.  
4. **Announce with reason** — outcomes delivered, costs, enhanced service — not apology.  
5. **Expect some churn** — if raise is correct, remaining base is healthier.  
6. **Measure** — close rate, churn, profit per customer, capacity.

## Operator checklist

- [ ] Value Equation review complete  
- [ ] New price set for new buyers  
- [ ] Existing customer policy written  
- [ ] Sales scripts updated for price objection  
- [ ] Pricing playbook structural plays considered  
- [ ] KPI watch window (30–90 days)
