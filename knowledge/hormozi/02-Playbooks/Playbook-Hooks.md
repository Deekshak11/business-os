# Playbook — Hooks

## When to use

- Ads/content get impressions but not attention  
- Outreach ignored  
- Creative tests fail at the first 1–3 seconds  
- Stage 2+ advertising volume needs higher conversion of eyeballs  

## Core definitions

- **Hook:** whatever people see/hear **first** (first seconds of video, first line, first image/sound).  
- **Job of the hook:** make a **micro-sale** — sell *attention*. Make them think “this is for me.”  
- **If the hook fails, nothing else matters** (offer, price, body copy never get seen).

## How hooks work (two parts)

1. **Call-out** — cut through noise (cocktail party effect: your “name” in the noise).  
2. **Condition for value** — promise: if *you* (target) do *this* (consume), you get *that* (reward). Can be explicit or implicit.

## Verbal hook types

| Type | Pattern |
|------|---------|
| Labels | Avatar identity words (“Local business owners…”) |
| Questions | Yes-questions / open questions |
| Conditionals | If you’re X and Y, you’re doing Z wrong |
| Commands | “Read this if…” |
| Statements | “The smartest thing you can do today…” |
| Lists/steps | “28 ways to…” |
| Narratives | Cold open story |
| Exclamations / provocative | Strong emotion / bold claim |

## Nonverbal hooks

Visuals and sounds that force a turn-of-head (drop/impact vs “important news” tone). Combine verbal + nonverbal when the platform allows.

## Process principles

- Specific enough for the **right** people; broad enough to get **many** of them.  
- Spend disproportionate time on hooks (example heuristic: if you have 10 hours to advertise, put ~8 into hooks).  
- **70-20-10 innovation rule** for hook R&D: mostly proven winners, some adjacent variants, small pure experiments.  
- One sentence change can flip a flop to a winner (attention framing > ego framing).

## Operator checklist

1. Write 20–50 call-outs for the avatar.  
2. Pair each with a value condition (explicit if/then when possible).  
3. Test hooks with fixed meat/CTA when possible.  
4. Kill losers fast; multiply winners (see Goated Ads kaleidoscope).  
5. Apply same craft to emails, reels, ads, cold openers.
