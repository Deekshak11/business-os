# Frameworks and Types Index

## YAML `type` values

| type | Meaning | Prefer for |
|------|---------|------------|
| `framework` | Laws, models, principles | Diagnosis & strategy |
| `structure` | Blueprints, formulas for form | Draft architecture |
| `process` | Step-by-step workflows | Execution order |
| `checklist` | Audit lists | QC before ship |
| `swipe` | Examples/templates | Inspiration (adapt) |
| `moc` | Maps / indexes | Navigation |
| `navigation` | OS how-to | Onboarding |

## High-signal frameworks (by name)

- Schwartz **5 States of Awareness** → `Market-*`
- Schwartz **5 Stages of Sophistication** → `Market-*`
- Cialdini **7 triggers** → `Laws-*`
- Kahneman **System 1/2 + heuristics** → `Laws-*`
- Hormozi **Value Equation** (in CopyOS offer docs) → `Offers-*`
- Masterson **6 lead types** / **4 Pillars** → `Headlines-Leads-*`, `Structures-*`
- Bencivenga **bullet types** → `Bullets-*`
- Brunson **False Beliefs / Perfect Webinar / Value Ladder** → `Objections-*`, `Structures-*`, `Funnels-Email-*`
- Testing hierarchy / LTV:CAC → `Optimization-*`
