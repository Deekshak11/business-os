# Query Router — CopyOS

## By task

| User task | Open first | Then |
|-----------|------------|------|
| “What is CopyOS / how do I use it?” | `00-Navigation/Nav-00-Index.md` | Workflows in Navigation |
| Emergency copy ASAP | Navigation workflows (Workflow #0) | Offers + Headlines + Structure |
| Why does persuasion work? | `01-Strategic-Foundation/Laws-*` | Specific Cialdini/Kahneman sections |
| Who is the customer / awareness? | `Market-*` (States, Stages, Avatar) | Campaign brief sections |
| Big Idea / USP / voice | `Positioning-*` | Mechanism, touchstone |
| Build the offer | `Offers-*` | Value Equation, stack, guarantee, urgency |
| Which format (VSL, letter, webinar)? | `Structures-*` | Specific blueprint section |
| Write headlines | `Headlines-Leads-*` | 4 U’s, formulas, swipes in Reference |
| Write lead / opening | `Headlines-Leads-*` (lead types) | Match to State |
| Write bullets / fascinations | `Bullets-*` | Reference bullet swipes |
| Proof / testimonials | `Proof-*` | Authority stack |
| Word choice / power words | `Language-*` + `Reference-*` | Clarity / hypnotic sections |
| Funnel map | `Funnels-Email-*` | Value ladder, funnel type |
| Email sequence | `Funnels-Email-*` (Soap Opera, Nurture, etc.) | Timing sections |
| Handle objections / FAQ | `Objections-*` | By category + SPIN + techniques |
| Low conversion / what to test | `Optimization-*` | Testing hierarchy, bottlenecks |
| Swipes / templates / checklists | `Reference-*` | Adaptation process (not blind copy) |
| B2B / high-ticket / platform specs | `Optimization-*` context parts | Proof + Language as needed |
