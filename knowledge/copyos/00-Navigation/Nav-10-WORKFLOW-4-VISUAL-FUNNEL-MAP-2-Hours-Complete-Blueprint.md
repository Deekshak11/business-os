## **WORKFLOW #4 — VISUAL FUNNEL MAP (2 Hours — Complete Blueprint):**

```
OBJECTIVE: A fully designed funnel (stages, conversion %, pages, sequences, metrics — total visual clarity)

HOUR 1 — STRUCTURE (Choose a Blueprint from Doc 10 Part II):

00-20min: Choose the funnel type (8 blueprints in Doc 10):
• Lead Gen (if the goal is email capture)
• Tripwire (for a low-ticket $27-297 offer converting cold traffic)
• VSL (for an info product in the $500-5k range in Brazil, Ladeira-style)
• Webinar (for high-ticket $1k-10k with live scarcity)
• [Choose based on goal/product/State]

20-60min: Map the stages (on paper/whiteboard/a digital tool like Lucidchart):
• Stage 1: Traffic source (Facebook ad, Native, Google — where they come from)
• Stage 2: Landing page (opt-in, tripwire, VSL — the first touchpoint)
• Stage 3: Email sequence (which type? How many emails? Timing?)
• Stage 4: Core offer page (VSL, sales letter, webinar — the main pitch)
• Stage 5: Checkout (upsell, order bump, payment)
• Stage 6: Post-purchase (onboarding, ascension, retention)

Draw: A flowchart (boxes for stages, arrows for flow, estimated conversion % on each arrow from benchmarks in Doc 12 Section 1.4)

───────────────────────────────────────────────────

HOUR 2 — METRICS + SEQUENCES (Doc 10 Part VII + Doc 12):

00-30min: Add metrics to each stage (benchmarks from Doc 12 Section 1.4):
• Ad → Landing: 2% CTR (1,000 impressions → 20 clicks)
• Landing → Opt-in: 30% (20 → 6 opt-ins)
• Sequence → Offer: 12% (6 → 0.72 customers)
• Revenue: 0.72 × $1,997 = $1,438
• Cost: $50 for ads (CPM of $20, 1,000 impressions = $20, + $30 for tools/ops = $50 total)
• Profit: $1,388
• ROI: 27.8:1 (a projection — validate with a real test)

30-60min: Attach sequences (templates from Doc 10):
• Email sequence: Which type? (Soap Opera for engagement, Nurture for education)
• Quantity: 5-10 emails (outline subjects + body bullets, don't write the full text yet — just map it out)
• Automation: Triggers (opt-in → sequence starts, link click → interest tag, purchase → onboarding)

OUTPUT AFTER 2H:
• Visual funnel map (a clear flowchart with stages/sequences/metrics/automation — a complete blueprint)
• Projection: Estimated metrics (to be validated with testing and adjusted with real data)
• Ready to: Execute (write the copy for the pages, emails, and set up the technical platform)

NEXT: Use Workflows #1-3-5 (you already have the headline and offer; write the emails, create the landing page) = create the funnel pieces using the map as a guide.
```
