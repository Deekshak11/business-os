## **SECTION 9.1 — CHECKLIST: 43 ITEMS (A Complete Audit)**

---

```
═══════════════════════════════════════════════════
CHECKLIST — UNIVERSAL LAWS IN COPY (A PRE-PUBLICATION QC)
═══════════════════════════════════════════════════

┌─ THE 3 AXIOMS (The Foundation) ─────────────────────┐
│ ☐ Is the copy salesmanship? (is a sales structure present)│
│ ☐ A pre-existing desire? (Intensity 7+, validated) │
│ ☐ Are shortcuts activated? (doesn't require deep S2 analysis)│
└──────────────────────────────────────────────────────┘

┌─ THE 7 TRIGGERS (Use a min of 3-4) ──────────────────┐
│ ☐ RECIPROCITY: Value given BEFORE? (LM, bonuses, 90/10)│
│ ☐ COMMITMENT: Micro-yeses? (a ladder, a quiz, text input)│
│ ☐ SOCIAL PROOF: Have others done it? (testimonials, numbers)│
│ ☐ LIKING: Is it human? (a voice, vulnerability, a story)│
│ ☐ AUTHORITY: Is it credible? (a 7-element stack, logos)│
│ ☐ SCARCITY: Is the urgency REAL? (a deadline, not fake)│
│ ☐ UNITY: "We" vs. "They"? (tribal, an enemy)       │
│ SCORE: __/7 (Aim: 3-4 at a minimum)                │
└──────────────────────────────────────────────────────┘

┌─ SYSTEM 1/2 + HEURISTICS ─────────────────────────┐
│ ☐ Is it simple? (8th-grade, no jargon)             │
│ ☐ Is S1 activated? (a story, emotion, visuals)     │
│ ☐ Specific numbers? (8,347, not ~8k)               │
│ ☐ A vivid story? (availability)                    │
│ ☐ Anchoring? (a high price before the real one)    │
│ ☐ A positive frame? (75% lean, not 25% fat)        │
│ ☐ The Endowment Effect? ("YOUR new X")             │
│ ☐ Limited choices? (2-3 max)                       │
└──────────────────────────────────────────────────────┘

┌─ COMPLEMENTARY FRAMEWORKS ─────────────────────────┐
│ ☐ Is "Free" used? (the power of zero)              │
│ ☐ The SUCCESs score? (aim for 5-6/6)               │
│ ☐ The One Sentence? (4-5/5 elements)               │
│ ☐ Is an LF8 drive identified? (1-2 minimum)        │
└──────────────────────────────────────────────────────┘

IF 35+/43 are CHECKED (80%+) → THE COPY IS SOLID (launch)
IF <35 → REVISE (principles are missing)

═══════════════════════════════════════════════════
```
