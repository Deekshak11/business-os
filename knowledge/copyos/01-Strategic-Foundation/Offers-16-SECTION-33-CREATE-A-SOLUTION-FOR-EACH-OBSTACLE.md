## **SECTION 3.3 — CREATE A SOLUTION FOR EACH OBSTACLE**

---

### **THE PROCESS:**

For EACH obstacle (the 15-30 you listed) → Create a SOLUTION (core product or bonus).

**No:** Ignore some (hoping the customer "figures it out" — they won't, they'll get frustrated, and they'll refund).

**Yes:** Address ALL of them (a comprehensive solution).

---

### **SOLUTION ASSIGNMENT:**

**Core product:** Solves the main obstacles (3-5 critical ones, the core value proposition).

**Bonuses:** Solve the secondary/tertiary obstacles (the 10-15+ remaining).

---

**Example (Weight Loss Program):**

```
OBSTACLES → SOLUTIONS:

┌─────────────────────────────────────────────────┐
│ Obstacle                │ Solution │ Where  │Cost You│Value Them│
├──────────────────────────┼─────────┼────────┼────────┼──────────┤
│1. Don't know what to eat │Meal plans│Bonus 1│$0 (digital)│R$300│
│2. No time to cook     │Quick recipes <15min│Bonus 2│$0│R$200│
│3. Hate the gym              │Home workouts│CORE│(core)│(core)│
│4. Tried before, failed   │Results guarantee│Guarantee│Risk│R$500 perceived│
│5. No accountability     │Group coaching calls│Bonus 3│R$50/month│R$1,000│
│6. Confused (info overload)│Step-by-step program│CORE│(core)│(core)│
│7. Fear of yo-yo effect    │Maintenance protocol│Bonus 4│$0│R$400│
│8. Don't know exercises     │Video demonstrations│CORE│(time to create)│(core)│
│9. Family not supportive      │Family guide (involve)│Bonus 5│$0│R$150│
│10. Stress eating (work)  │Stress management module│Bonus 6│$0│R$250│
│[...]│[...]│[...]│[...]│[...]│
└─────────────────────────────────────────────────┘

TOTAL: Core (solves 3-5 main ones) + 6 Bonuses (solve 10-15 others)

Cost to YOU: ~$50/month (group calls cost split among 20-50 people) + time to create (one-time)

Value to THEM: Core + R$2,800 in bonuses = R$4,800+ total perceived
```

---

**HORMOZI'S PRINCIPLE (Ideal Bonuses):**

**HIGH perceived value** for the customer (R$100-500+ each).

**LOW cost** to you (R$0-50 to deliver).

**MARGIN:** R$450+ of perceived value for a R$50 real cost = **9:1+ margin** (sustainable, profitable scaling).

---

**HIGH value / LOW cost bonuses (examples):**

✅ **Digital products:** Ebooks, templates, recordings, checklists (one-time creation cost, $0 to deliver infinitely, perceived value of R$50-300+ each)

✅ **Group coaching/calls:** Cost split among 20-50 participants (R$50/person cost), perceived value of R$500-1,000 (access to coaching)

✅ **Software/tool access:** Marginal cost to deliver (hosting), subscription perceived as valuable (R$50-200/month if standalone)

✅ **Community access:** $0 cost after setup (platform), perceived value of R$200-800 (networking, support, belonging)

✅ **Email/chat support:** Cost is time (manageable if systematized with FAQs, response templates), value of R$300-500 (peace of mind, not alone)

---

**Bonuses to AVOID (low margin):**

❌ **High-cost physical products:** Supplements ($50-100/customer cost), printed books ($15-30 each) — margins get tight.

❌ **Your 1-on-1 time:** 1-on-1 calls (time = doesn't scale, burnout), custom work (not replicable) — ok for VERY high-ticket, bad for info products.

❌ **Third-party subscriptions:** Paying an ongoing fee ($50-200/customer/month) — eats into margins, not sustainable at scale.

---

**Stick to:** Digital (R$0 scaling cost), group (cost is split), access (marginal cost).
