## **SECTION 3.2 — 4 TYPES OF MECHANISMS**

**Words:** 700

**Categorization:**

**Type A — Ingredient:** "It contains X, which does Y"  
Example: "Green tea extract (EGCG) increases fat oxidation"

**Type B — Process:** "It works via N steps"  
Example: "A 3-Step Validation System (Presell → Build → Scale)"

**Type C — Discovery:** "It's based on research/breakthrough Z"  
Example: "The Johns Hopkins protocol targets the psoas (the root cause)"

**Type D — System:** "It uses framework/method W"  
Example: "The SPIN Selling framework adapted for copywriting"

Each type: 175 words, 2 examples, when to use.
