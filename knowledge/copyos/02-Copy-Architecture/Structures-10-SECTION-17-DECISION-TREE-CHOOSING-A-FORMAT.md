## **SECTION 1.7 — DECISION TREE: CHOOSING A FORMAT**

---

### **INPUTS (From Doc 2 — The Diagnosis):**

**Input #1 — SCHWARTZ STATE (1-5):**

Where is the client on their awareness journey?

State 1 (Most Aware): Knows the product, wants it
State 2 (Product-Aware): Compares options
State 3 (Solution-Aware): Researches solutions
State 4 (Problem-Aware): Feels the pain, doesn't know of solutions
State 5 (Unaware): Doesn't know they have a problem

---

**Input #2 — PRICE (Product Ticket Size):**

Low: <R$500
Medium: R$500-2,000
High: R$2,000-10,000
Very High: R$10,000+

---

**Input #3 — GOAL (Campaign Objective):**

**Direct sale:** Close the sale immediately (a complete transaction)
**Lead gen:** Capture email/info (to nurture later)
**High-ticket:** Application → Call → Close (multi-step)
**Engagement:** Viewing/reading (content, not an immediate sale)

---

### **DECISION TREE (Flowchart):**

```
┌─────────────────────────────────────────────────┐
│ START: Complete diagnosis (Doc 2)               │
│ • State: [1-5]                                  │
│ • Price: [Low/Med/High/Very High]               │
│ • Goal: [Sale/Lead/High-ticket/Engagement]      │
└─────────────────────────────────────────────────┘
              ↓
         ┌────────────┐
         │ STATE?     │
         └────┬───────┘
              │
    ┌─────────┼─────────┬─────────┐
    │         │         │         │
  State    State     State     State
   1-2      3        4-5     (High-ticket,
(Aware)  (Solution) (Problem/  any State)
                    Unaware)      │
    │         │         │         │
    ↓         ↓         ↓         ↓
          
STATE 1-2 (Most/Product-Aware):
├─ Price <R$500: EMAIL (short, direct offer)
├─ Price R$500-2k: SALES LETTER (medium, 3-6k words) or VSL (30min)
└─ Price R$2k+: VSL (45min) or WEBINAR

STATE 3 (Solution-Aware):
├─ Goal Lead Gen: LANDING PAGE (lead magnet)
├─ Goal Sale <R$2k: SALES LETTER (6-8k words) or VSL (30-45min)
└─ Goal Sale R$2k+: VSL (45min) or PERFECT PROMO (8-12k words)

STATE 4-5 (Problem/Unaware):
├─ Cold traffic: ADVERTORIAL → VSL (2-step funnel)
├─ Goal Engagement: PERFECT PROMO (long, educational, 10k+ words)
├─ Goal Sale High: WEBINAR (60-90min, education first)
└─ Goal Lead Gen: LANDING PAGE (curiosity/story hook)

HIGH-TICKET (R$10k+, Any State):
└─ ALWAYS: Application/Survey → Call → Close
   Copy: Landing page (for the application) + Email sequence (to nurture)
   
SPECIAL CASES:
├─ Native Ads (Taboola): ADVERTORIAL (800-1,200w, editorial style)
├─ Facebook/IG Ads: AD CREATIVE (15-90sec, GOATED framework)
└─ Email List (Existing): EMAIL SEQUENCE (Soap Opera, Nurture)
```

---

**Decision Tree OUTPUT:** The recommended format (based on objective inputs).

**Not:** "I'll use a sales letter because I like them" (personal preference).

**Yes:** "State 4 + Price R$1,500 + Goal of a direct sale = a 45-min VSL is recommended (enough education for State 4, the pricing justifies the length, and the goal is a direct sale)" (data-driven).

---

**Flexibility:** These recommendations are not absolute (A/B test formats when the budget allows).

**BUT:** It's an informed starting point (not a random guess).

---

**Validation:**

After choosing a format (from the Decision Tree), reference historical data:

"State 4 products (problem-aware) using a 45-min VSL: Typical conversion is 2-6% (Ladeira/Agora data)."

"State 1 using a short email: Typical conversion is 8-15% (existing customers)."

**Benchmarks:** Help calibrate expectations (know what "good" looks like for the chosen format).
