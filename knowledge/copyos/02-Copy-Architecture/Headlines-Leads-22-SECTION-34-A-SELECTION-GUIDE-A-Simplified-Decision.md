## **SECTION 3.4 — A SELECTION GUIDE (A Simplified Decision)**

```
═══════════════════════════════════════════════════
A HEADLINE SELECTION GUIDE (Quick Reference)
═══════════════════════════════════════════════════

INPUTS (From the Doc 2 Diagnosis):
1. State: [1-5]
2. Stage: [1-5]
3. Price: [Low/Med/High]

OUTPUT: Recommended headline types (the top 3 formulas to use)

────────────────────────────────────────────────────

IF State 1-2 (Aware):
→ Direct Benefit, Offer, Guarantee
→ Short (10-25w), transactional, urgency

IF State 3 (Solution):
→ How-To, Number, Reason-Why
→ Medium (20-40w), educational, mechanism

IF State 4-5 (Problem/Unaware):
→ Story, Question, Secret
→ Long (30-60w), narrative, curiosity

────────────────────────────────────────────────────

IF Stage 1-2 (Virgin/Low Competition):
→ Direct claims are OK (not saturated)
→ Simplicity wins (elaboration is unnecessary)

IF Stage 3-4 (Mechanisms):
→ Reveal the HOW (differentiation via the mechanism)
→ An authority boost (studies, experts)

IF Stage 5 (Saturation):
→ Story/Experience ONLY (claims are ignored)
→ Identification > Promise

────────────────────────────────────────────────────

IF Price <$500:
→ Short headlines (they don't justify long copy)
→ Direct benefit, offer, urgency

IF Price $500-5,000:
→ Medium/long headlines (education is needed)
→ Layers of mechanism, authority, and social proof

IF Price >$5,000:
→ Long, comprehensive headlines (to justify the investment)
→ Multi-element (authority+mechanism+proof+guarantee stacked)

═══════════════════════════════════════════════════
```
