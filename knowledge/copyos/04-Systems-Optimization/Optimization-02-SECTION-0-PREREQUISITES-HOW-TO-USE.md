# **SECTION 0 — PREREQUISITES & HOW TO USE**

---

### **Prerequisites:**

**→ CRITICAL REQUIREMENT: A complete copy launched** (or about to be — you need baseline metrics).

**→ USEFUL:**
- Doc 4 (Offer) — for testing offer elements
- Doc 6 (Headlines) — for testing headline variations
- Doc 10 (Funnels) — for the defined funnel metrics

**Optional:** Docs 5-9 (copy elements — to be tested after the offer/headlines/funnels are optimized).

---

### **Time Investment:**

**Reading:** 3-4h (for the testing philosophy, methodologies, specializations, and benchmarks).

**Application:** **Perpetually ongoing** (testing is never-ending):
- Weekly: 2-4h (to set up 1 test, analyze the previous one, and implement the winner)
- Monthly: 4-8h (to review the aggregate results, plan the next tests, and document the learnings)
- Quarterly: 8-12h (for major overhauls based on 3 months of accumulated data)

**Total for the first year:** ~150-200h (of continuous testing — an investment that is justified, as conversion compounds by +30-100% in the first year of optimization).

---

### **Expected Output:**

✅ **A testing calendar:** With weekly tests scheduled (a minimum of 52/year).

✅ **Established baseline metrics:** (before you optimize — to know your starting point).

✅ **An improvement log:** Documented (with the results of each test, the learnings, and the implemented winners).

✅ **A chosen specialization:** Context-specific (B2B, High-Ticket, BR, etc — with the copy adapted appropriately).

**Doc 12 IMPROVES the copy** (testing identifies winners, iteration refines it, and specialization adapts it to different contexts).
